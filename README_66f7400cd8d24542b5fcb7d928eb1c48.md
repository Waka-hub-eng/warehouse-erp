# 海外仓ERP系统 - Vercel部署版

## 🚀 快速部署到Vercel

### 1. 准备工作
- 注册 [GitHub](https://github.com) 账号
- 注册 [Vercel](https://vercel.com) 账号  
- 准备 Supabase 项目信息

### 2. 部署步骤

#### 步骤1: 创建GitHub仓库
1. 在GitHub创建新仓库 `warehouse-erp`
2. 上传所有项目文件到仓库

#### 步骤2: 连接Vercel
1. 登录 [Vercel控制台](https://vercel.com/dashboard)
2. 点击 "New Project"
3. 选择您的GitHub仓库
4. 点击 "Deploy"

#### 步骤3: 配置环境变量
在Vercel项目设置中添加环境变量：
```
VITE_SUPABASE_URL=https://your-project.supabase.co
VITE_SUPABASE_ANON_KEY=your-anon-key
```

#### 步骤4: 重新部署
配置环境变量后，触发重新部署即可。
